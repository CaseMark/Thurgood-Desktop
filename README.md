# Thurgood Desktop

AI-powered legal research assistant built on [OpenCode](https://github.com/sst/opencode).

## Installation

```bash
curl -fsSL https://thurgood.law/install | bash
```

## About

Thurgood is an AI legal assistant developed by [Case.dev](https://case.dev). This desktop application provides an intelligent interface for legal research and document analysis.

## Development

```bash
# Install dependencies
bun install

# Run development server
bun dev

# Build
bun run --cwd packages/opencode build -- --single
```
